# Download building-level data from MCDS (current format, 2018+)

Downloads the Building Demographic Data report from Missouri DESE's MCDS
SSRS system. The report is exported to Excel format.

## Usage

``` r
download_mcds_building_data(end_year)
```

## Arguments

- end_year:

  School year end

## Value

Data frame with building-level enrollment data
